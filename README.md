# To-Do-List-Aplikasi-Daftar-Tugas-menggunakan-Penyimpanan-Lokal
Project To-Do List adalah aplikasi web sederhana yang membantu pengguna mencatat dan mengelola tugas harian secara teratur. Fitur utamanya meliputi menambah, mengedit, menandai selesai, dan menghapus tugas, sehingga pengguna dapat memantau aktivitas dengan lebih mudah.
